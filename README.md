# Concentric-Square

### Built With

- ReactJS
- React-hooks

# Getting Started

### Prerequisites:

- NodeJS-npm

### Local Installation

1. Clone this repo and install the npm dependencies.\
   You can folow the commands to do the same.

```bash
git clone https://github.com/WahidX/test-app.git
cd test-app
npm install
```

2. Now We are good to go. Just run the below command.

```bash
npm start
```
